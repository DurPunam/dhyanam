# 🧘‍♂️ Dhyanam – Meditation Website for Students

Dhyanam is a modern, responsive meditation website designed specifically for students.  
It provides calming content, guided breathing, meditation music, mindful games, and quick stress-relief tools.

---

## 🌿 Features

- 🎯 Hero section with calming introduction
- 🧘 5-Minute Relief section for instant stress reduction
- 🎵 Meditation Music section
- 🎮 Relaxing Mind Games section
- 🌬 Breathing Practice section
- 🔗 Visual storytelling layout using SVG curve connectors
- 📱 Fully responsive design
- 🎨 Pure HTML & CSS (No JavaScript)

---

## 🛠 Technologies Used

- HTML5
- CSS3
- Flexbox Layout
- SVG Graphics
- CSS Animations
- Responsive Design (Media Queries)

## 🎨 Design Concept

The layout alternates images left and right to create visual storytelling.  
SVG curved connectors visually link each section, guiding the user's eye naturally down the page.

The design focuses on:
- Calm color palette
- Clean typography
- Smooth animations
- Good readability

---

## 📱 Responsive Behavior

- Desktop → Two-column alternating layout with connectors
- Mobile → Stacked layout with connectors hidden for better readability
