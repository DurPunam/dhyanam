# 🧘‍♂️ Dhyanam – Meditation Website for Students

Dhyanam is a modern, accessible, and responsive meditation website designed specifically for students.  
It provides calming content, guided breathing animations, meditation music, mindful games, and quick stress-relief tools with a beautiful UI/UX.

---

## 🌿 Features

- 🎯 Hero section with calming introduction and smooth animations
- 🧘 5-Minute Relief section for instant stress reduction
- 🎵 Meditation Music section with interactive play/pause toggle
- 🎮 Relaxing Mind Games section
- 🌬 **Breathing Practice with animated breathing circle** - visual guide that expands/contracts
- 🔗 Visual storytelling layout using animated SVG curve connectors
- 📱 Fully responsive mobile-first design
- 🌙 **Dark mode toggle** for comfortable viewing in any lighting
- ♿ **Enhanced accessibility** with ARIA labels, semantic HTML, and keyboard navigation
- 🎨 Calming color palette (soft greens, blues, neutrals)
- ✨ Smooth scrolling and subtle hover animations

---

## 🛠 Technologies Used

- HTML5 (Semantic markup with ARIA)
- CSS3 (Custom properties, animations, gradients)
- Flexbox Layout
- SVG Graphics
- CSS Keyframe Animations
- Responsive Design (Mobile-first with media queries)
- JavaScript (Dark mode toggle, music controls)
- Google Fonts (Poppins)

## 🎨 Design Concept

The layout alternates images left and right to create visual storytelling.  
SVG curved connectors visually link each section, guiding the user's eye naturally down the page.

The design focuses on:
- **Calm color palette** - Soft greens (#66BB6A, #81C784), peaceful blues, and neutral tones
- **Clean typography** - Poppins font family for modern readability
- **Smooth animations** - Breathing circle, SVG path drawing, fade-ins, hover effects
- **Excellent readability** - 1.6 line-height, optimal character width, high contrast
- **Accessibility-first** - Semantic HTML, ARIA labels, keyboard navigation, reduced motion support
- **Dark mode** - Eye-friendly theme switching with localStorage persistence

---

## 📱 Responsive Behavior

- **Desktop (>768px)** → Two-column alternating layout with animated SVG connectors
- **Tablet (≤768px)** → Stacked layout with adjusted spacing
- **Mobile (≤480px)** → Single column, connectors hidden, optimized touch targets
- All breakpoints use relative units (em) for better scaling
